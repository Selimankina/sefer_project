# Sefer Project — распознавание номеров на изображениях

Приложение для автоматического распознавания номеров на табличках с изображений и последующего переименования файлов.

## Возможности

- Полностью локальная работа (без интернета)
- Групповая обработка изображений из папки
- Поддержка формата `.jpeg / .jpg`
- Распознавание номеров:
  - от 1 до 4 цифр
  - возможна одна латинская буква в конце (`a–e`)
- Автоматическое переименование файлов:
  - дополнение номера до 4 разрядов (`7 → 0007`)
  - обработка дубликатов (`0025.jpg`, `0025_1.jpg`, ...)
- Обработка сложных случаев:
  - низкая уверенность → префикс `!_`
  - номер не найден → файл не переименовывается + `!_`

---

## Как это работает

Приложение использует многоступенчатый pipeline обработки изображения:

1. **ROI Detection**
   - поиск области с номером

2. **Preprocessing**
   - подготовка изображения
   - нормализация
   - обрезка ROI (области с табличкой)

3. **Digits Detection**
   - детекция отдельных цифр

4. **Post-processing**
   - сборка номера
   - фильтрация
   - сортировка цифр
   - формирование итогового номера

7. **Renaming**
   - генерация нового имени файла
   - обработка конфликтов
